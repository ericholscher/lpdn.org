Lawrence Programmers Drinking Night

Drinkers with a programming problem

LPDN is a weekly gettogether for tech-interested people in Lawrence, KS.

It generally happens every tuesday at 7PM. Check the #lawrence channel on irc.freenode.net for more information

Check out the site on Github and fork liberally.
